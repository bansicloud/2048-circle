2048 Circle!

By: Kira L
Use the arrow keys to move the tiles. If you are on a device without a keyboard, this game will not work. When two tiles with the same number collide, they will merge into their sum. Try to get the 2048 tile!




This game is a variation of 2048, which can be played at play2048.co. If you like this, check out my other projects at kiraprograms.com!
